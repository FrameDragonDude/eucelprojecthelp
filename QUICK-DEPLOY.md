# 🌐 TẠO ĐƯỜNG DẪN CHO WEBSITE EUCEL

## ⚡ CÁCH NHANH NHẤT (3 PHÚT)

### 1️⃣ Build website
```bash
npm run build
```

### 2️⃣ Chọn 1 trong 3 cách:

---

## 🎯 CÁCH 1: NETLIFY DROP (ĐỀ XUẤT - KHÔNG CẦN CODE)

**Cực kỳ đơn giản, chỉ kéo thả!**

1. Chạy file `deploy.bat` (double-click)
2. Truy cập: https://app.netlify.com/drop
3. Kéo folder `build` vào trang web
4. ✅ **XONG!** Nhận link như: `https://eucel-abc123.netlify.app`

---

## 🚀 CÁCH 2: VERCEL (NHANH NHẤT)

```bash
# Bước 1: Cài Vercel
npm install -g vercel

# Bước 2: Deploy
vercel --prod
```

👉 Làm theo hướng dẫn, nhấn Enter để chọn mặc định

✅ **XONG!** Nhận link như: `https://eucel.vercel.app`

---

## ⚡ CÁCH 3: NGROK (DEMO TẠM THỜI)

**Dùng khi cần chia sẻ ngay, không cần deploy vĩnh viễn**

```bash
# Bước 1: Chạy website
npm start

# Bước 2: Mở terminal mới, chạy ngrok
ngrok http 3000
```

✅ Nhận link tạm thời: `https://abc123.ngrok.io`

**LƯU Ý**: Link này chỉ hoạt động khi máy bạn đang bật!
